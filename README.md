# Qi V2 Agent

Local AI coding assistant with RAG-powered codebase understanding.

## Architecture

- **Frontend**: Ink + TypeScript CLI
- **Workflow**: LangGraph (macro) + n8n (micro)
- **LLM**: LangChain + Ollama (local)
- **Knowledge**: ChromaDB vector database
- **Tools**: MCP servers
